## 1. 主题概述

物流与供应链Schema主题涵盖**从GS1到EDI**
的物流供应链标准化Schema体系，是物流
管理和供应链集成的基础。

### 1.1 主题范围

- **GS1 Schema**：GS1标准（GTIN、GLN、SSCC、EPCIS）
- **EDI Schema**：EDI标准（EDI X12、EDIFACT）（规划中）

### 1.2 核心价值

- **标准化**：基于GS1、EDI等国际标准
- **可追溯性**：支持供应链全程追溯
- **互操作性**：支持不同系统间的数据交换
- **形式化**：数学形式化定义

## 4. 标准对标

### 4.1 国际标准

- **GS1标准**：全球统一标识系统标准
- **EDI标准**：电子数据交换标准

### 4.2 行业标准

- **GTIN**：全球贸易项目代码
- **GLN**：全球位置码
- **SSCC**：系列货运包装箱代码
- **EPCIS**：EPC信息服务标准

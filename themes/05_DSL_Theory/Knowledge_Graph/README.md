## 1. 概述

知识图谱用于表示DSL Schema转换中的知识，
包括实体、关系、属性等，支持知识推理和应用。

### 1.1 核心价值

- **知识表示**：结构化知识表示
- **知识推理**：基于图谱的知识推理
- **知识应用**：转换指导和质量评估

---

## 2. 核心概念

### 2.1 知识图谱定义

**知识图谱**是一个有向图 `KG = (E, R, A)`，其中：

- `E`：实体集合
- `R`：关系集合
- `A`：属性集合

### 2.2 Schema知识图谱

```text
Schema_KG = {
  Entities: {Schema, Type, Constraint, ...},
  Relations: {has_type, has_constraint, ...},
  Properties: {name, value, ...}
}
```

---

## 3. 知识表示

### 3.1 实体定义

**Schema实体**：

- **Schema**：Schema定义实体
- **Type**：类型实体
- **Constraint**：约束实体
- **Field**：字段实体

### 3.2 关系定义

**Schema关系**：

- **has_type**：Schema有类型
- **has_constraint**：类型有约束
- **has_field**：Schema有字段
- **transforms_to**：Schema转换到

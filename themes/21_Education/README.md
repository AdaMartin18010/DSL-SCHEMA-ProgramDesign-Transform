# 教育行业Schema主题

## 📊 主题概述

教育行业Schema主题涵盖学习管理系统、学生信息系统、在线教育平台等领域的Schema定义。

## 📁 Schema列表

1. **LMS_Schema**（学习管理系统）
   - 标准：SCORM、xAPI、IMS LTI
   - 状态：✅ 已完成

2. **SIS_Schema**（学生信息系统）
   - 标准：PESC、SIF、Ed-Fi
   - 状态：⏳ 进行中

3. **Online_Education_Schema**（在线教育平台）
   - 标准：xAPI、IMS Common Cartridge
   - 状态：⏳ 规划中

## 📚 文档结构

每个Schema包含5个标准文档：

- `01_Overview.md` - 概述文档
- `02_Formal_Definition.md` - 形式化定义
- `03_Standards.md` - 标准对标
- `04_Transformation.md` - 转换体系（含PostgreSQL存储）
- `05_Case_Studies.md` - 实践案例

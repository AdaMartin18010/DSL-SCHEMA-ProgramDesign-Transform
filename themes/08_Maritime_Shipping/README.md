## 1. 主题概述

海运与航运Schema主题涵盖**从船舶管理到
货物追踪**的海运航运标准化Schema体系，是
海运物流和航运管理的基础。

### 1.1 主题范围

- **Maritime Schema**：海运与航运标准（船舶信息、货物信息、航线信息、港口信息）

### 1.2 核心价值

- **标准化**：基于IMO、ISO等国际标准
- **可追溯性**：支持货物全程追溯
- **互操作性**：支持不同系统间的数据交换
- **形式化**：数学形式化定义

## 4. 标准对标

### 4.1 国际标准

- **IMO标准**：国际海事组织标准
- **ISO 28000**：供应链安全管理标准
- **ISO 28005**：电子港口清关标准

### 4.2 行业标准

- **EDIFACT**：电子数据交换标准
- **UN/LOCODE**：联合国地点代码标准
- **IMO船舶编号**：国际海事组织船舶编号标准

## 1. 标准体系概述

安全标准Schema标准体系分为三个层次：

1. **ISO标准**：ISO 27001、ISO 27002等
2. **NIST标准**：NIST Cybersecurity Framework、NIST SP 800-53等
3. **OWASP标准**：OWASP Top 10、OWASP ASVS等

---

## 2. ISO 27001规范

### 2.1 ISO/IEC 27001:2022

**标准名称**：ISO/IEC 27001:2022
**核心内容**：

- 信息安全管理体系要求
- 114个控制措施
- 风险评估方法

**Schema支持**：完整支持
**参考链接**：<https://www.iso.org/standard/27001>

### 2.2 ISO/IEC 27002:2022

**标准名称**：ISO/IEC 27002:2022
**核心内容**：

- 信息安全控制措施指南
- 控制措施实施指南

**Schema支持**：完整支持

## 3. NIST规范

### 3.1 NIST Cybersecurity Framework 1.1

**标准名称**：NIST Cybersecurity Framework 1.1
**核心内容**：

- 5个核心功能
- 23个类别
- 108个子类别

**Schema支持**：完整支持
**参考链接**：<https://www.nist.gov/cyberframework>

### 3.2 NIST SP 800-53

**标准名称**：NIST SP 800-53
**核心内容**：

- 安全控制措施
- 控制措施基线
- 控制措施评估

**Schema支持**：完整支持

## 4. OWASP规范

### 4.1 OWASP Top 10

**标准名称**：OWASP Top 10
**核心内容**：

- Web应用安全风险
- 风险描述和缓解措施

**Schema支持**：完整支持
**参考链接**：<https://owasp.org/www-project-top-ten/>

### 4.2 OWASP ASVS

**标准名称**：OWASP Application Security Verification Standard
**核心内容**：

- 应用安全验证标准
- 安全控制要求

**Schema支持**：完整支持

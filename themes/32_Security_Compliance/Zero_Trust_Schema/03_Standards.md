## 1. 标准体系概述

零信任Schema标准体系分为两个层次：

1. **NIST规范**：NIST零信任架构规范
2. **相关标准**：CISA、OAuth等

---

## 2. NIST Zero Trust规范

### 2.1 NIST SP 800-207

**标准名称**：NIST SP 800-207 Zero Trust Architecture
**核心内容**：

- 零信任原则
- 零信任架构组件
- 实施指南

**Schema支持**：完整支持
**参考链接**：<https://csrc.nist.gov/publications/detail/sp/800-207/final>

### 2.2 NIST零信任架构指南

**标准名称**：NIST Zero Trust Architecture Guide
**核心内容**：

- 零信任实施步骤
- 架构设计指南
- 最佳实践

**Schema支持**：完整支持

---

## 3. 相关标准

### 3.1 CISA零信任成熟度模型

**标准名称**：CISA Zero Trust Maturity Model
**核心内容**：

- 零信任成熟度评估
- 实施路径

**Schema支持**：完整支持

### 3.2 OAuth 2.0

**标准名称**：OAuth 2.0
**核心内容**：

- 零信任使用OAuth 2.0进行身份认证
- 授权流程

**与零信任的关系**：

- 零信任架构使用OAuth 2.0进行身份认证
- OAuth 2.0是零信任身份验证的基础

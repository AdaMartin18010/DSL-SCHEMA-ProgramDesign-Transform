## 1. 主题概述

智慧城市Schema主题涵盖**从IoT到城市管理**
的智慧城市标准化Schema体系，是智慧城市
建设和城市数字化转型的基础。

### 1.1 主题范围

- **Smart City Schema**：智慧城市标准（IoT设备、传感器、城市数据、城市服务）

### 1.2 核心价值

- **标准化**：基于ISO、IEC等国际标准
- **可扩展性**：支持智慧城市各领域扩展
- **互操作性**：支持不同系统间的数据交换
- **形式化**：数学形式化定义

## 4. 标准对标

### 4.1 国际标准

- **ISO 37120**：城市可持续发展指标
- **ISO/IEC 30141**：物联网参考架构
- **IEC 61850**：智能电网通信标准

### 4.2 行业标准

- **OneM2M**：物联网服务层标准
- **LoRaWAN**：低功耗广域网标准
- **NB-IoT**：窄带物联网标准

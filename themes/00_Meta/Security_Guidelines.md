# 安全指南
## Security Guidelines

**版本**: 2.2.0  
**最后更新**: 2026-02-17

---

## 安全最佳实践

### 1. API安全

- 使用HTTPS/TLS 1.3
- API Key轮转周期: 90天
- 速率限制: 1000请求/分钟/IP
- JWT令牌过期: 1小时

### 2. 数据安全

- 敏感字段加密 (AES-256)
- 数据库连接使用SSL
- 定期备份 (每日)
- 备份保留期: 30天

### 3. 代码安全

- 依赖项自动扫描
- 代码审查强制要求
- Secret检测 pre-commit
- 容器镜像扫描

### 4. 部署安全

- 最小权限原则
- 网络隔离
- 日志审计
- 入侵检测

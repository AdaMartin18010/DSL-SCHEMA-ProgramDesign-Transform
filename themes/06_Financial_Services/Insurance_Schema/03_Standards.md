## 1. 标准体系概述

保险业务Schema标准体系分为六个层次：

1. **数据交换标准**：ACORD等保险数据交换标准
2. **偿付能力标准**：Solvency II、C-ROSS等偿付能力监管标准
3. **会计标准**：IFRS 17、US GAAP等保险合同会计标准
4. **数据保护标准**：GDPR、数据安全法等客户数据保护标准
5. **精算标准**：精算实务标准、准备金评估标准
6. **行业标准**：各险种特定的行业标准

```
┌─────────────────────────────────────────────────────────────────┐
│                      保险业务标准体系                            │
├───────────────┬───────────────┬───────────────┬─────────────────┤
│  数据交换标准  │  偿付能力标准  │  会计标准      │   数据保护标准  │
├───────────────┼───────────────┼───────────────┼─────────────────┤
│ ACORD         │ Solvency II   │ IFRS 17       │ GDPR            │
│ CSIO          │ C-ROSS        │ US GAAP       │ 数据安全法      │
│ CDA           │ SAM           │ Local GAAP    │ 个人信息保护法  │
│ DAR           │ LICAT         │ IFRS 9        │ 保险数据安全    │
└───────────────┴───────────────┴───────────────┴─────────────────┘
```

## 2. ACORD标准

### 2.1 ACORD概述

**组织全称**：Association for Cooperative Operations Research and Development

**成立时间**：1970年

**核心使命**：制定保险行业数据交换标准

**标准覆盖范围**：

- 财产意外险（P&C）
- 寿险与年金（Life & Annuity）
- 再保险（Reinsurance）
- 保证险（Surety）

**Schema支持**：完整支持

**参考链接**：
[ACORD官网](https://www.acord.org/)

### 3.5 偿付能力资本要求（SCR）

**标准公式法**：

```
SCR = Basic Solvency Capital Requirement + Operational Risk - Adjustment

BSCR = √[Σ(i,j) Corr(i,j) × SCRi × SCRj]

其中:
- SCR_mkt: 市场风险模块
- SCR_def: 违约风险模块
- SCR_life: 寿险承保风险模块
- SCR_nl: 非寿险承保风险模块
- SCR_health: 健康险承保风险模块
```

**风险模块分解**：

| 风险模块 | 子风险 | 说明 |
|----------|--------|------|
| **市场风险** | 利率风险 | 利率变动影响 |
| | 权益风险 | 股价波动 |
| | 房地产风险 | 房产价值波动 |
| | 利差风险 | 信用利差变动 |
| | 货币风险 | 汇率变动 |
| | 集中度风险 | 单一风险暴露 |
| **信用风险** | 交易对手违约 | 再保/衍生交易对手 |
| | 投资违约 | 债券等投资违约 |
| **承保风险** | 死亡率风险 | 实际vs假设死亡率 |
| | 长寿风险 | 年金赔付超预期 |
| | 失效率风险 | 保单失效差异 |
| | 费用风险 | 费用超预期 |
| | 巨灾风险 | 累积巨灾损失 |

### 3.6 最低资本要求（MCR）

**MCR计算**：

$$
MCR = \max(MCR_{calculated}, MCR_{absolute\_floor})
$$

$$
MCR_{calculated} = \min(SCR \times 25\%, SCR \times 45\%)
$$

**绝对底线**：

| 业务类型 | 绝对底线（欧元） |
|----------|-----------------|
| 非寿险（不含信用/保证险） | 2,500,000 |
| 寿险 | 3,700,000 |
| 再保险（非寿险） | 3,600,000 |
| 再保险（寿险） | 3,600,000 |

## 4. IFRS 17标准

### 4.1 IFRS 17概述

**标准名称**：IFRS 17 Insurance Contracts

**发布机构**：国际会计准则理事会（IASB）

**核心目标**：

1. 提供保险合同的统一会计模型
2. 提高保险合同的透明度和可比性
3. 及时反映保险合同的经济实质
4. 提供更相关的财务信息

**生效日期**：2023年1月1日（全球同步实施）

**适用范围**：

- 签发保险合同的主体
- 持有的再保险合同
- 具有相机参与分红特征的投资合同

### 4.2 保险合同识别

**保险合同定义**：

```
合同在约定日期满足以下条件时属于保险合同：
1. 合同约定的保险事项对保单持有人产生不利影响
2. 该不利影响转移给签发人（保险公司）
3. 转移的不利影响并非微不足道
```

**重大保险风险测试（GMM适用）**：

| 情形 | 测试结果 | 适用模型 |
|------|----------|----------|
| 存在重大保险风险 | 保险合同 | GMM/PAA/VFA |
| 无重大保险风险 | 服务合同/投资合同 | IFRS 9/15 |
| 相机参与分红 | 投资合同 | IFRS 17 |

### 4.3 保险合同合并与分拆

**合同合并**：

```
同一保单持有人同时满足以下条件的合同应合并：
1. 同一时间或相近时间订立
2. 整体作为一揽子商业方案
3. 合同整体盈亏取决于其他合同赔付
```

**合同分拆**：

```
┌─────────────────────────────────────────────────────────────────┐
│                      保险合同分拆                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  嵌入式衍生工具 ─────────────────────────────────── IFRS 9      │
│  (可单独计量且与主合同不紧密相关)                                │
│                                                                 │
│  可明确区分的投资成分 ─────────────────────────── IFRS 9        │
│  (可单独计量，投资回报不依赖于保险事项)                          │
│                                                                 │
│  可明确区分的商品/服务承诺 ────────────────────── IFRS 15       │
│  (可单独计量，保单持有人可从其他供应商获得)                      │
│                                                                 │
│  剩余部分 ─────────────────────────────────────── IFRS 17      │
│  (保险合同成分)                                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.4 计量模型

**通用模型（GMM - General Measurement Model）**：

```
┌─────────────────────────────────────────────────────────────────┐
│                      GMM计量模型                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  保险合同负债 = 履约现金流 + 合同服务边际 - 亏损部分             │
│                                                                 │
│  ┌──────────────────────────────────────────────────────────┐  │
│  │  履约现金流 (Fulfilment Cash Flows)                       │  │
│  │                                                           │  │
│  │  未来现金流估计 (Estimates of Future Cash Flows)         │  │
│  │  ├─ 保费收入预测                                          │  │
│  │  ├─ 理赔支出预测                                          │  │
│  │  ├─ 费用支出预测                                          │  │
│  │  └─ 其他现金流                                            │  │
│  │                                                           │  │
│  │  货币时间价值及金融风险 (Discounting)                     │  │
│  │  └─ 使用反映现金流量特征的折现率                          │  │
│  │                                                           │  │
│  │  非金融风险风险调整 (Risk Adjustment for Non-Fin Risk)   │  │
│  │  └─ 反映非金融风险的不确定性                              │  │
│  └──────────────────────────────────────────────────────────┘  │
│                                                                 │
│  + 合同服务边际 (Contractual Service Margin)                   │
│  │   └─ 代表未来未实现利润                                   │
│  │       CSM ≥ 0 (初始确认时)                                │
│  │       CSM在保险期间内摊销                                 │
│  │                                                           │
│  │  摊销基础：覆盖范围单位 (Coverage Units)                  │
│  │  ├─ 保险期间                                              │
│  │  ├─ 保额金额                                              │
│  │  ├─ 预期给付金额                                          │
│  │  └─ 保单数量                                              │
│  │                                                           │
│  │  当期摊销 = 期初CSM × (当期CU / 预计总CU)                 │
│  │                                                           │
│  │  CSM调整：                                                │
│  │  ├─ 与未来服务相关的履约现金流变动                         │
│  │  ├─ 汇率变动影响                                          │
│  │  └─ 汇率风险变动影响 (选择OCI时)                          │
│  │                                                           │
│  │  亏损合同处理：                                            │
│  │  ├─ 亏损部分立即确认为费用                                │
│  │  └─ 后续履约现金流变动处理                                │
│  └──────────────────────────────────────────────────────────┘  │
│                                                                 │
│  - 亏损部分 (Loss Component)                                   │
│     └─ 履约现金流为负时确认                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**保费分配法（PAA - Premium Allocation Approach）**：

适用条件：
1. 保险期间不超过12个月
2. 或能合理预计GMM与PAA结果差异不重大

**可变费用法（VFA - Variable Fee Approach）**：

适用条件（需同时满足）：
1. 合同具有相机参与分红特征
2. 相机参与分红金额重大
3. 相机参与分红支付时间由企业相机决定

**三模型对比**：

| 特征 | GMM | PAA | VFA |
|------|-----|-----|-----|
| 适用条件 | 默认 | 短期/简化 | 分红险 |
| CSM计算 | 公式计算 | 简化为未赚保费 | 公式计算 |
| 金融风险 | 可选择OCI | 固定 | 必须OCI |
| 分红处理 | 履约现金流 | 不适用 | 可变对价 |
| 复杂度 | 高 | 低 | 中 |

### 4.5 保险合同负债

**负债组成**：

```
┌─────────────────────────────────────────────────────────────────┐
│                  IFRS 17保险合同负债                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  未到期责任负债 (Liability for Remaining Coverage)             │
│  ├─ 履约现金流（与未来服务相关部分）                           │
│  ├─ 合同服务边际                                             │
│  └─ 亏损部分（如有）                                          │
│                                                                 │
│  + 已发生赔款负债 (Liability for Incurred Claims)             │
│  ├─ 履约现金流（与已发生保险事项相关部分）                     │
│  └─ 不包括合同服务边际                                        │
│                                                                 │
│  = 保险合同负债总额                                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.6 列报与披露

**财务状况表列报**：

| 项目 | 说明 |
|------|------|
| 保险合同资产 | 分出再保险合同资产 |
| 保险合同负债 | 未到期责任+已发生赔款 |
| 分出再保险合同资产 | 摊回分保准备金 |
| 分出再保险合同负债 | 再保负债（如有） |

**损益表列报**：

```
保险服务收入
  ├─ 预计赔付及费用
  ├─ 保险获取现金流量摊销
  ├─ 已发生赔款负债相关履约现金流变动
  ├─ 亏损部分及亏损转回
  ├─ CSM摊销
  └─ 其他服务收入

保险服务费用
  ├─ 实际赔付及费用
  ├─ 保险获取现金流量资产减值
  └─ 其他服务费用

投资成分（列示为收入扣减项）

承保财务业绩 = 保险服务收入 - 保险服务费用
```

**披露要求**：

| 披露类别 | 内容 |
|----------|------|
| **数量信息** | 合同数量、保额、保费等 |
| **金额信息** | 履约现金流、CSM、亏损部分变动分析 |
| **假设与判断** | 主要假设、敏感性分析 |
| **风险信息** | 保险风险、金融风险敞口 |

## 5. 其他重要标准

### 5.1 C-ROSS（中国风险导向偿付能力体系）

**体系名称**：China Risk-Oriented Solvency System

**核心内容**：

| 支柱 | 内容 | 说明 |
|------|------|------|
| **第一支柱** | 定量资本要求 | 实际资本、最低资本、偿付能力充足率 |
| **第二支柱** | 定性监管要求 | SARMRA评估、风险管理能力 |
| **第三支柱** | 市场约束机制 | 信息披露、公开透明 |

**偿付能力充足率**：

$$
\text{偿付能力充足率} = \frac{\text{实际资本}}{\text{最低资本}} \times 100\%
$$

**监管标准**：

| 指标 | 充足I类 | 充足II类 | 充足III类 | 不足 |
|------|---------|----------|-----------|------|
| 综合偿付能力充足率 | 100-150% | 150-250% | ≥250% | <100% |
| 核心偿付能力充足率 | 50-75% | 75-100% | ≥100% | <50% |

### 5.2 GDPR与数据保护

## 7. 标准发展趋势

### 7.1 2024-2025年趋势

#### 7.1.1 IFRS 17实施深化

- **趋势**：保险公司持续优化IFRS 17实施
- **影响**：需要更完善的保险合同Schema定义
- **关键领域**：
  - CSM计算自动化
  - 假设更新流程
  - 披露报告优化

#### 7.1.2 Solvency II审查与修订

- **趋势**：2023年审查结果推动标准修订
- **影响**：偿付能力计算和报告要求变化
- **潜在变化**：
  - 长期担保措施审查
  - 审慎再评估（VA）修订
  - 报告要求简化

#### 7.1.3 气候风险纳入标准

- **趋势**：气候风险纳入偿付能力和会计标准
- **影响**：需要新的风险计量和披露要求
- **发展方向**：
  - 气候情景分析
  - 物理风险与转型风险
  - 绿色保险标准

### 7.2 2025-2026年展望

#### 7.2.1 数字保险标准发展

- **趋势**：数字化保险产品需要新标准
- **影响**：参数保险、嵌入式保险等新形态
- **发展方向**：
  - API标准化
  - 实时数据交换
  - 智能合约标准

#### 7.2.2 数据标准化深化

- **趋势**：保险数据标准更加统一
- **影响**：跨公司、跨国数据交换更加便利
- **发展方向**：
  - 数据湖标准
  - 实时数据流
  - AI/ML数据标准

#### 7.2.3 ESG信息披露要求

- **趋势**：ESG信息强制披露扩大至保险
- **影响**：投资端和承保端ESG数据需求
- **发展方向**：
  - 承保ESG风险披露
  - 投资碳足迹披露
  - 气候相关财务披露（TCFD）

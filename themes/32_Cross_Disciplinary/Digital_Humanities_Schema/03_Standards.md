## 1. 标准概述

数字人文Schema标准对标涵盖**数字人文领域的国际标准和行业标准**，包括TEI、IIIF、CIDOC-CRM等文本、图像、概念模型标准。

**标准覆盖**：

- **文本标准**：TEI（文本编码倡议）
- **图像标准**：IIIF（国际图像互操作框架）
- **概念模型**：CIDOC-CRM（概念参考模型）

---

## 2. 国际标准

### 2.1 TEI标准

**标准名称**：Text Encoding Initiative (TEI)

**标准组织**：TEI Consortium

**标准版本**：TEI P5（2024年）

**核心内容**：

- XML文本编码标准
- 文本结构标记
- 语义标注
- 元数据标准

**标准特点**：

- ✅ 完整的文本编码规范
- ✅ 语义标注支持
- ✅ 广泛采用

**采用率**：⭐⭐⭐⭐⭐（5/5）

### 2.2 IIIF标准

**标准名称**：International Image Interoperability Framework (IIIF)

**标准组织**：IIIF Consortium

**标准版本**：IIIF 3.0（2024年）

**核心内容**：

- 图像API标准
- 图像查看器标准
- 图像搜索标准
- 图像呈现标准

**标准特点**：

- ✅ 图像互操作性
- ✅ 标准化API
- ✅ 广泛采用

**采用率**：⭐⭐⭐⭐⭐（5/5）

### 2.3 CIDOC-CRM标准

**标准名称**：CIDOC Conceptual Reference Model

**标准组织**：CIDOC

**标准版本**：CIDOC-CRM 7.1（2024年）

**核心内容**：

- 文化遗产概念模型
- 实体关系模型
- 语义互操作性

**采用率**：⭐⭐⭐⭐（4/5）

## 3. 行业标准

### 3.1 Dublin Core标准

**标准名称**：Dublin Core Metadata Initiative (DCMI)

**核心内容**：

- 元数据标准
- 15个核心元素
- 扩展元素

**采用率**：⭐⭐⭐⭐⭐（5/5）

### 3.2 EAD标准

**标准名称**：Encoded Archival Description (EAD)

**核心内容**：

- 档案描述标准
- XML编码
- 层次化结构

**采用率**：⭐⭐⭐⭐（4/5）

## 4. 标准对比分析

| 标准 | 类型 | 应用场景 | 采用率 |
|------|------|---------|--------|
| **TEI** | 文本编码 | 文本数字化 | 95%+ |
| **IIIF** | 图像框架 | 图像互操作 | 90%+ |
| **CIDOC-CRM** | 概念模型 | 文化遗产 | 80%+ |
| **Dublin Core** | 元数据 | 资源描述 | 95%+ |

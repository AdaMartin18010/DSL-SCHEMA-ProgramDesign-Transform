# 决策树导航工具

**版本**: v1.0  
**创建日期**: 2026-02-15  

---

## 功能概述

决策树导航工具是一个交互式工具，帮助用户根据决策树图快速找到合适的Schema设计决策。

### 主要功能

1. **交互式决策导航**
   - 基于问答的决策引导
   - 支持所有主题的决策树
   - 智能路径推荐

2. **决策记录**
   - 保存决策历史
   - 决策理由记录
   - 导出决策报告

3. **可视化展示**
   - 决策路径可视化
   - 选项对比展示
   - 决策树浏览

---

## 使用方法

```bash
# 启动交互式导航
python decision_navigator.py

# 指定主题导航
python decision_navigator.py --theme 01_Industrial_Automation

# Web界面
python decision_navigator.py --web
```

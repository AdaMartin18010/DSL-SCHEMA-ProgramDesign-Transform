## 3. 标准体系

- **3GPP**：第三代合作伙伴计划标准
- **ETSI NFV**：网络功能虚拟化标准
- **O-RAN**：开放无线接入网标准
- **eTOM**：增强的电信运营图
- **ITIL**：IT基础设施库
- **TM Forum**：电信管理论坛标准
- **SNMP**：简单网络管理协议
- **NETCONF**：网络配置协议
- **YANG**：数据建模语言

---

## 4. 应用场景

- 电信运营支撑系统
- 网络设备管理
- 网络性能监控

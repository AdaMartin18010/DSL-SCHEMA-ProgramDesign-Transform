## 1. 转换场景

### 1.1 EDI到GS1转换

**场景**：物流行业到零售行业的数据交换。

**转换规则**：

- EDI段 → GS1元素
- EDI代码 → GS1标识符
- EDI日期 → GS1日期格式

### 1.2 HL7到FHIR转换

**场景**：医疗行业内部标准转换。

**转换规则**：

- HL7段 → FHIR资源
- HL7代码 → FHIR编码
- HL7日期 → FHIR日期时间

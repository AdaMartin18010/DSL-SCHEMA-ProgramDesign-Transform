## 1. 静态类型系统

### 1.1 类型检查

**编译时类型检查**：

- 在编译时检查类型错误
- 提前发现类型问题
- 提高代码质量

### 1.2 类型推断

**自动类型推断**：

- 编译器自动推断类型
- 减少类型声明
- 提高开发效率

## 3. 类型转换

### 3.1 隐式转换

**自动类型转换**：

- 编译器自动转换类型
- 简化代码编写
- 可能引入类型错误

### 3.2 显式转换

**手动类型转换**：

- 开发者明确指定转换
- 更安全的转换
- 代码更清晰

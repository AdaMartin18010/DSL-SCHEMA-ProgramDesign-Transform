## 📝 文档分类

### 计划文档（plans/）

包含项目的各种计划文档，如：
- 全面推进计划
- 内容质量改进计划
- 知识图谱增强计划
- 批判性评价改进计划

### 报告文档（reports/）

包含项目的各种报告文档，如：
- 完成报告
- 进度报告
- 状态报告
- 验证报告

### 分析文档（analysis/）

包含项目的各种分析文档，如：
- 批判性分析
- 网络权威信息整合

### 路线图文档（roadmaps/）

包含项目的实施路线图文档。

### 增强文档（enhancements/）

包含项目的增强计划文档。

### 整合文档（integration/）

包含项目的整合文档。

## 🔍 查找文档

如需查找特定文档，请使用以下方式：

1. **按类型查找**：根据文档类型在对应目录查找
2. **按关键词查找**：使用文件搜索功能
3. **查看索引**：查看 `archive/reports/completion_reports/ARCHIVE_COMPLETION_REPORTS.md`

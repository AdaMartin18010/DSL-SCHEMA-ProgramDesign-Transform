## 2. 三大目录导航

### 2.1 themes/ 目录：主题实践

**定位**：28个主题、81个Schema的完整文档集

**快速入口**：

- 📖 [主题总览](./themes/README.md)
- 📑 [文档索引](./themes/DOCUMENT_INDEX.md)
- 🔍 [快速参考](./themes/QUICK_REFERENCE.md)

**主题分类**：

- **基础技术**（01-05）：工业自动化、IoT、物理设备、编程转换、DSL理论
- **行业应用**（06-24）：金融服务、物流、医疗、教育、农业等19个行业
- **企业级**（25-28）：AI+Code集成、企业财务、数据分析、绩效管理

**Schema文档结构**（每个Schema包含5个文档）：

- `01_Overview.md` - 概述
- `02_Formal_Definition.md` - 形式化定义
- `03_Standards.md` - 标准对标
- `04_Transformation.md` - 转换体系
- `05_Case_Studies.md` - 实践案例

## 7. 使用建议

### 7.1 首次访问

1. **阅读本文档**（5分钟）- 了解项目结构
2. **阅读快速入门**（5分钟）- 了解统一框架
3. **阅读三大目录整合**（10分钟）- 了解目录关系
4. **选择感兴趣的路径**深入阅读

### 7.2 日常使用

- **查找主题**：使用 `themes/README.md` 或 `themes/DOCUMENT_INDEX.md`
- **查找理论**：使用 `view/ai_prompt.md` 或 `view/theory/`
- **查找框架**：使用 `structure/FRAMEWORK_NAVIGATION.md`
- **查找关系**：使用 `structure/GLOBAL_THEME_RELATIONSHIP_ANALYSIS.md`

### 7.3 深入研究

- **理论研究者**：从 `structure/UNIFIED_LOGIC_FRAMEWORK.md` 开始
- **实践应用者**：从 `view/practices/` 开始
- **关系研究者**：从 `structure/GLOBAL_THEME_RELATIONSHIP_ANALYSIS.md` 开始

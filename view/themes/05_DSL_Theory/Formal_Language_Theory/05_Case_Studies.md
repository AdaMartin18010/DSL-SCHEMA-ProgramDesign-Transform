## 2. 案例1：JSON Schema语法分析

### 2.1 场景描述

**应用场景**：
使用形式语言理论分析JSON Schema的语法结构。

### 2.2 Schema定义

**JSON Schema定义**：

```json
{
  "type": "object",
  "properties": {
    "id": {"type": "integer"},
    "name": {"type": "string"}
  }
}
```

### 2.3 语法分析

**分析结果**：

- **文法类型**：上下文无关文法
- **非终结符**：Schema, Object, Properties, Field, Type
- **终结符**：object, integer, string, id, name

### 2.4 语法树构建

**语法树结构**：

```text
Schema
└── Object
    └── Properties
        ├── Field(id, Type(integer))
        └── Field(name, Type(string))
```

### 3.2 Schema定义

**OpenAPI定义**：

```yaml

  schemas:
```

✅ 语义正确
✅ 类型一致
